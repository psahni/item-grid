# READ ME

## Clone the repository
#### git clone git@github.com:psahni/item-grid.git

## Ruby Version
#### 2.1.0
#### rvm install 2.1.0

## Database
#### Make sure you have mysql5 installed in your system

## Go to Project root
#### cd /your-path/item-grid

## Install Gems
#### bundle install

## Create Database
#### rake db:create

## Migrate Database
#### rake db:migrate

## Insert sample data
#### rake db:seed

## Start the server
#### rails server


## Go to Browser
#### http://localhost:3000
